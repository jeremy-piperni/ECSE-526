README for main.py (A1 Power Connect 4, by Jeremy Piperni)

Variables that can be changed by the user:

connect_to_server: True = program will connect to the game server for the power connect 4 tournament
		   False = program will run in command line (see AI_vs_AI variable)
game_id: game id for the power connect 4 tournament
playing_color: wanted playing color of the AI
TCP_IP: ip for power connect 4 tournament server
TCP_PORT: port for power connect 4 tournament server
AI_vs_AI: True = simulates a game of the AI playing white vs AI playing black
	  False = makes the user be able to input moves and play against the AI
depth: select depth of state expansion for the AI
